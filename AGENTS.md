# AGENTS.md

Guidance for Codex when working in this repository.

Read `docs/agent-rules.md` first.
Read `docs/current-state.md` before work.

Repo workflow:
- Read only files relevant to the task unless needed
- Prefer small focused changes over broad repo scans

Project state workflow:
- Update `docs/current-state.md` only after meaningful changes
- Keep only current truth in:
  - Current systems
  - Known issues
  - Next 3 priorities
  - Files that matter most
- Replace stale bullets instead of appending logs or history
- Do not store changing app state in `AGENTS.md`
